# Zack's Mohawk Limited
## QuickDiff

## Overview

Command line tool for quick comparison of two files

## How To Install

	npm install

## How To Configure

No configuration required.

## How To Setup Command Line Alias

To be able to run QuickDiff from any folder on Mac, run the following setup script.

	./setup_mac.sh

For Linux machines, run the following setup script.

	./setup_linux.sh

## How To Run

From any location, type the following:

	quickdiff
